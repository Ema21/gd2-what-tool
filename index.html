About this project

This project started because I like pens. And because I have a serious pet-peeve when designers don't know the difference between broad-edge calligraphy versus pointed-pen calligraphy. Especially those learning typography!!

I don't claim to know everything about every pen ever made, and I can certainly think of more rare or specialty pens that weren't mentioned here, but I hope this website is a good starting point.
